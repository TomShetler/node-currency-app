# Currency App

This is a simple node based app to convert currencies. It makes use of the **yargs** and **request** modules.

##To run this program:

1. Clone/download the source files
2. Run `npm install` to download dependencies
3. Run `node app.js -c USD`
4. The example uses USD as the base currency but you can use any abbreviation you'd like to convert
